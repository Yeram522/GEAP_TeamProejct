## 📣 무서운 건 싫으니까 마을 구조에 올인 하려고 합니다. 소개

하루 아침에 몬스터에게 점령당한 마을..
이젠 마을에 그 누구도 보이지 않는다.

" 무서운 건 싫어... 혼자 남겨지는 건 더 싫어...! 내가 직접 마을을 구해내고 모든 것을 원래대로 되돌려 놓겠어!"
